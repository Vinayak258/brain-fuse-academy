# Brain Fuse Academy

Premium multi-page coaching institute website built for **Brain Fuse Academy** using **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

## 🌐 Overview

Brain Fuse Academy is a modern educational website designed for SSC/GSEB coaching students.
The website focuses on:

* Premium educational branding
* Fast and responsive UI
* Multi-page architecture
* Student achievements & results showcase
* Admission enquiry system
* Mobile-first design
* Professional coaching institute experience

---

## ✨ Features

* Modern multi-page React architecture
* Responsive premium UI
* Framer Motion animations
* Sticky animated navbar
* Results & toppers showcase
* Courses & batch information
* Gallery section
* Contact & admission enquiry
* WhatsApp integration
* Optimized for mobile & desktop

## 🚀 Run Locally

Clone the project:

```bash
git clone https://github.com/Vinayak258/brain-fuse-academy.git
```

Go to project folder:

```bash
cd brain-fuse-academy
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
